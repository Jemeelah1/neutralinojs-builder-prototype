# Neutralinojs Builder Prototype

This project is a prototype CLI plugin for generating platform-specific installers in Neutralinojs.

## Purpose
This prototype explores how a unified builder command can simplify the packaging process for developers.

## Features
- CLI command parsing
- Modular target-based architecture
- Dynamic loading of build targets

## Example Usage
node builder.js nsis --x64

## Note
This is an early prototype created as part of my GSoC 2026 proposal for Neutralinojs.
